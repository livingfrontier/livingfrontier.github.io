#+title: Redis

Redis 是基于内存的键值数据库，是 [[id:56bb7eba-e269-441a-a4d6-fa5abf4c26af][NoSQL]] 运动中的代表项目。

基于内存的意思是它在默认情况下，将数据保存在内存中而不是外存中。但是内存数据库虽然相比于外存性能占优，但是却具有易失性。不过其实不用过于担心，Redis 如果运行在服务器上，那么就不需要担心频繁重启导致数据丢失，此外，Redis 一般会用作传统外存数据库的缓存，起加速外存数据库，比如说 [[id:2227f095-7c94-412b-b62d-3275d6564aba][MySQL]] 的作用。如下图所示：
